How Shift Works

Shift operates as a super lightweight desktop launcher that connects your workspaces directly to your OS terminal daemon.

01

1. Create Workspace

Define a workspace profile and populate it with the key native apps, website URLs, and absolute folder paths that your project requires.

02

2. Configure Global Shortcuts

Assign a custom, system-wide key shortcut (e.g., Ctrl + Alt + Space) to summon the Workspace Launcher from anywhere.

03

3. Launch Instantly

Hit your global shortcut, search for your workspace, and hit Enter. Shift launches everything in parallel in less than two seconds.

04

4. Stay Ready

Shift stays active quietly in your system tray as a minimal OS background daemon, waiting for your next context trigger.

Core Engine Capabilities

Unified developer tools, streamlined workflows.

Stop switching focus. Shift is engineered purely to restore your workspace instantly with zero over-engineering.

Instant Workspace Launch

Launch multiple applications, local directory paths, and browser URLs concurrently in seconds.

Dual Global Shortcut System

Separate global control shortcuts for summoning the instant Workspace Launcher and accessing the full Dashboard configuration.

Background Tray Daemon

Runs quietly in your OS system tray. Capitalizes on minimal RAM footprint (< 15MB) for global keybindings and instant launches.

Native OS Integration & App Discovery

Full compatibility with Linux, macOS, and Windows. Automatically detects natively installed OS applications to speed up workspace creation.

Interactive Setup & Folder Browser

Scaffolds new configurations easily with an interactive onboarding wizard and an OS-native absolute directory picker to prevent typos.

Workspace Import & Export

Export your environment groups as `.shift` files. Share workspace profiles with your team or backup your setup effortlessly.

Developer Quick-Start

Get running inside 60 seconds

Shift is completely open-source and bootstraps in simple terminal prompts.

STEP 1

Clone the official repository:

git clone https://github.com/DzarelDeveloper/Shift.git
STEP 2

Enter the directory and fetch core packages:

cd Shift && npm install
STEP 3

Boot the development server with Tauri:

npm run tauri:dev
No hidden telemetry. All data is stored completely locally on your machine.
