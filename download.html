CLI version installation

📦 Instalasi CLI

Instal Shift secara instan lewat satu baris perintah terminal di Linux, macOS, maupun Windows.

Quick Install (Recommended)

Bash One-Liner

Jalankan one-liner ini untuk menginstall Shift secara otomatis:

$ curl -fsSL https://raw.githubusercontent.com/DzarelDeveloper/Shift/main/install.sh | bash

Quick Install (Recommended)

PowerShell

Buka PowerShell as Administrator dan jalankan:

Langkah 1: Set Execution Policy
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
Langkah 2: Ambil & Jalankan Script
Invoke-Expression (Invoke-WebRequest -Uri "https://raw.githubusercontent.com/DzarelDeveloper/Shift/main/install.ps1" -UseBasicParsing).Content
Troubleshooting: Error "git is not recognized"

Jika Anda mendapatkan error yang menyebutkan The term 'git' is not recognized saat instalasi di Windows, itu berarti Git belum terinstal. Anda dapat menginstalnya dengan menjalankan perintah berikut di PowerShell:

winget install --id Git.Git -e --source winget

Setelah selesai, tutup PowerShell, buka kembali yang baru, dan jalankan ulang perintah instalasi di atas.

System Requirements

Verify that your machine supports running the Shift local terminal playground environment.

Windows

  • Windows 10 or higher
  • 64-bit Architecture (x64)
  • Windows PowerShell 5.1+

macOS

  • macOS 12+ (Monterey or higher)
  • Apple Silicon (M1/M2/M3) or Intel Core
  • Xcode Command Line Tools installed

Linux

  • Ubuntu 22.04+ or Debian 12+
  • Fedora 40+ or Arch Linux
  • Fuse Library support (for AppImages)

Verify Download Integrity

Users can verify SHA256 checksums to ensure downloaded installer files have not been modified or corrupted.

Run the following command in PowerShell to compute the SHA256 hash of the download:

certutil -hashfile Shift_0.6.8_x64-setup.exe SHA256

Expected Output matches: a7d8c910f1b2c3d4e5f60718293a4b5c6d7e8f90a1b2c3d4e5f60718293a4b5c

Run this command in the macOS Terminal app to inspect file integrity:

shasum -a 256 Shift_0.6.8_universal.dmg

Expected Output matches: d3e4f5a6b7c8d9e0f1a2b3c4d5e6f708192a3b4c5d6e7f8019aabcbccddeef00

Use the standard sha256sum compiler command in bash/zsh:

sha256sum Shift_0.6.8_amd64.AppImage

Expected Output matches: b8c9d0e1f2a3b4c5d6e7f80192a3b4c5d6e7f80192a3b4c5d6e7f80192a3b4c5
