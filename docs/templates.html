Workspace Templates

Shift supports Workspace Import & Export using .shift files. Below are official ready-to-use templates for each platform. Simply copy the JSON code, save it as a .shift file, then import it into Shift.

Important: Customize Your Paths

The templates below use standard application paths (absolute paths). Make sure the listed applications are actually installed on your computer at the specified paths. You can edit the paths after importing them into Shift.

Windows: Frontend Development Template

Opens VS Code, Chrome (Localhost), and File Explorer to your Documents folder.

How to Import a Template

  1. Copy one of the JSON templates above (choose the one that matches your operating system).
  2. Open Notepad or any text editor and paste the contents.
  3. Save the file with a .shift extension (e.g., my-template.shift).
  4. Open the Shift application → Go to the Settings tab → Click Import Workspaces and select your .shift file.
  5. The workspace will be created with all entries and is ready to launch immediately!
